How private transactions work on Solana

🔒 What is this?

A privacy tool for Solana that breaks the on-chain link between sender and recipient. Traditional Solana transfers create a direct, traceable connection between wallets — this tool eliminates that link using multiple techniques.

Every transaction is routed through disposable shadow wallets, token swaps, and private pools to make blockchain analysis extremely difficult.

⚡ How it works

Instead of sending funds directly from A → B, the tool creates an indirect path that breaks traceability:

Core Techniques

  • Shadow Wallets — Fresh, disposable wallets generated for each transaction. Your main wallet never touches the recipient.
  • Token Swaps — Funds are converted between tokens mid-route (e.g. SOL → USDC → SOL), breaking token-based tracking.
  • Private Pools — External liquidity pools that receive from one address and send from a completely different, unlinked address.
  • Amount Splitting — Large amounts are split into random smaller pieces, each taking a different path.
  • MEV Protection — Transactions use bundle protection to prevent front-running and sandwich attacks.

🛡️ Privacy Levels

Choose the level that matches your needs. Higher levels take longer but provide stronger unlinkability.

Enhanced ⭐⭐⭐
Shadow wallet + token swap. Funds go from your wallet to a fresh shadow, get swapped to a different token, then sent to the recipient. Fast (10–30 seconds).
Your Wallet → Shadow Wallet → Token Swap → Recipient
Maximum ⭐⭐⭐⭐⭐
Shadow wallet + private pool bridge. Funds enter a private pool from your shadow and exit from a completely unlinked address to the recipient. Takes 2–10 minutes.
Your Wallet → Shadow Wallet → Private Pool → Recipient
Maximum+ ⭐⭐⭐⭐⭐+
Double bridge with intermediate shadow. Funds pass through two separate private pools with a second shadow wallet in between — maximum unlinkability. Takes 5–20 minutes.
Your Wallet → Shadow 1 → Pool 1 → Shadow 2 → Pool 2 → Recipient

📊 Fees

  • Service Fee: 1% of the transaction amount
  • Network Fees: Standard Solana transaction fees (~0.000005 SOL per signature)
  • Shadow Wallet Gas: A small SOL amount is sent to the shadow wallet to cover its transaction fees (~0.003 SOL)
  • Private Pool Fee: The pool provider takes a small spread (typically 0.5–1%)

The total cost is displayed in the summary before you confirm the transaction.

🔄 Recovery System

If a transaction fails mid-way (network error, pool timeout, etc.), your funds are not lost. The app includes an automatic recovery system:

  • Automatic Detection: The app checks for stuck funds in shadow wallets on startup
  • Recovery Banner: If stuck funds are detected, a recovery banner appears at the bottom of the screen
  • One-Click Recovery: Click "Recover Funds" to send stuck funds back to your main wallet
  • Shadow Key Storage: Shadow wallet keys are stored locally in your browser until recovery is complete

⚠️ Do not clear your browser data while a transaction is in progress. Shadow wallet keys are stored in localStorage and are needed for recovery.

🔐 Security Notes

  • No custody: Your wallet is never connected to any third-party. All signing happens via your wallet extension (Phantom, Backpack, etc.)
  • Single approval: Each send requires only one wallet popup — all internal steps are pre-built into a single transaction where possible
  • Shadow keys are local: Disposable wallet keys are generated in-browser and never sent to any server
  • Open progress: Every step is shown in real-time in the console so you can verify what's happening

❓ FAQ

How long does a transaction take?

Enhanced: 10–30 seconds. Maximum: 2–10 minutes. Maximum+: 5–20 minutes. The private pool processing is the main factor.

Can the recipient see who sent the funds?

No. The recipient receives funds from a private pool address that has no connection to your wallet. On Maximum+, there are two pool hops making analysis even harder.

What if the transaction fails?

Funds stay in the shadow wallet. The recovery system will detect them and let you reclaim them with one click. No funds are ever lost.

Does this work with any Solana wallet?

Yes — any wallet that supports the Solana Wallet Standard (Phantom, Backpack, Solflare, etc.).

Why is there a minimum amount?

Private pools have minimum transaction sizes. Amounts below the minimum would be rejected by the pool. The app enforces these limits before attempting a transaction.
